About

sweep.js is a small JavaScript library (<10kb minified) that enables proper color transitions through the HSL and HUSL spaces. Ordinary CSS transitions or existing frameworks like Velocity.js convert HSL colors to RGB before transitioning. sweep.js addresses this by letting you transition through the color spectrum.

I've written an in-depth post about the need for HSL transitions here.

Use

Using sweep.js to transition an element's color is easy. All of its dependencies are bundled, so all you have to do is include the script.

<script src="path/to/sweep.js"></script>
